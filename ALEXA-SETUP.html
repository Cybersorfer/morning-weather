  1. Open Alexa Developer Console and sign in with your Amazon account.
  2. Create Skill → Name: My Morning Weather → Custom → Alexa-hosted (Node.js).
  3. Build → Interfaces → turn on Alexa Web API for Games.
  4. Build → Invocation → invocation name: my morning weather.
  5. Code tab → replace all code with the contents of alexa-skill/index.js in this folder (URL already set).
  6. Click Deploy → set skill to Development.
  7. Alexa app on phone → Skills & Games → Your Skills → Dev → enable My Morning Weather on the Fire tablet.

Test on tablet: "Alexa, open My Morning Weather"
